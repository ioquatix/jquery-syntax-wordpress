*** jQuery.Syntax WordPress Plugin ***

jQuery.Syntax is a light-weight client-side syntax highlighter, which dynamically loads external dependencies when required. It uses jQuery to make it cross-browser compatible and to simplify integration.

This is the WordPress Plugin. It includes jQuery-1.4.1.min.js and jQuery.Syntax. It works using the standard <pre class="syntax brush-LANGUAGE"> syntax.
